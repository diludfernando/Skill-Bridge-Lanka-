# Skill Bridge Design System

A comprehensive guide to the visual language of the Skill Bridge platform. We use the **"Trust & Connection"** theme to convey professionalism, reliability, and growth.

## üé® Color Palette

### Primary Colors (Brand)
Used for headers, primary actions, and key brand elements.
- **Deep Royal Blue:** `#0F172A` (`--color-primary`)
  - *Usage:* Main headers, footer background, primary text.
- **Primary Light:** `#1E293B` (`--color-primary-light`)
  - *Usage:* Secondary backgrounds, dark mode elements.

### Accent Colors (Action)
Used for calls-to-action (CTAs), links, and highlights.
- **Vibrant Sky Blue:** `#0EA5E9` (`--color-accent`)
  - *Usage:* Primary buttons, icons, active states.
- **Accent Hover:** `#0284C7` (`--color-accent-hover`)
  - *Usage:* Hover states for primary buttons.

### Neutral Colors (Structure)
Used for backgrounds, borders, and text hierarchy.
- **Background:** `#F8FAFC` (`--color-background`) - *Light Slate*
- **Surface:** `#FFFFFF` (`--color-surface`) - *White*
- **Text Main:** `#0F172A` (`--color-text-main`) - *Deep Blue/Black*
- **Text Muted:** `#64748B` (`--color-text-muted`) - *Slate Grey*
- **Border:** `#E2E8F0` (`--color-border`) - *Light Grey*
